Em algum momento, percebi que ter um único link para compartilhar — algo simples, direto e que reunisse tudo em um só lugar — seria extremamente útil. Conhecia o Linktree e ferramentas semelhantes, mas queria criar a minha própria versão, com minha identidade visual e total controle sobre o conteúdo.

## O Projeto

A página de links é uma aplicação simples no estilo linktree, com todos os meus links importantes organizados em um único lugar: site pessoal, blog, redes sociais e e-mail. O foco era manter algo minimalista e performante, sem dependências desnecessárias.

## Funcionalidades

As principais funcionalidades implementadas são:

- **Performance**: renderização estática com Next.js App Router;
- **UI limpa**: Tailwind CSS v4 e fonte Geist;
- **Links organizados**: site, blog, redes sociais e e-mail;
- **Foto de perfil**: imagem local otimizada com Next.js Image.

## Tecnologias Utilizadas

Para o desenvolvimento, utilizei:

- **Next.js 16**: framework React com App Router, responsável pela renderização estática;
- **TypeScript**: linguagem principal do projeto, garantindo tipagem estática;
- **Tailwind CSS 4**: framework de estilização utilitária;
- **Geist**: fonte do Google Fonts utilizada no projeto.
