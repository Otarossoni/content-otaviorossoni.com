Tener un sitio web personal siempre fue algo que quería, pero durante mucho tiempo lo pospuse. Cuando finalmente decidí crearlo, no quería solo una página estática con una plantilla genérica. Quería algo que reflejara mi identidad como desarrollador — minimalista, performante y multilingüe. Así nació otaviorossoni.com.

## El Proyecto

El sitio es un portafolio personal con blog integrado, soporte para Markdown con syntax highlighting e internacionalización para tres idiomas: portugués, inglés y español. La idea era que cualquier persona, independientemente del idioma, pudiera navegar por el contenido.

## Funcionalidades

Las principales funcionalidades implementadas son:

- **Internacionalización**: soporte para Portugués (PT), Inglés (EN) y Español (ES) con next-intl;
- **Performance**: renderizado estático con Next.js App Router + Turbopack;
- **Blog con MDX**: posts con GitHub Flavored Markdown, syntax highlighting y tipografía estilizada;
- **Modo oscuro/claro**: alternancia suave con preferencia guardada en el navegador;
- **UI limpia**: Tailwind CSS v4, Phosphor Icons, fuente Geist.

## Tecnologías Utilizadas

Para el desarrollo, utilicé:

- **Next.js 16**: framework React con App Router, responsable del renderizado estático y enrutamiento;
- **TypeScript**: lenguaje principal del proyecto, garantizando tipado estático;
- **Tailwind CSS 4**: framework de estilización utilitaria;
- **next-intl**: biblioteca de internacionalización para Next.js;
- **MDX + next-mdx-remote + rehype-highlight**: stack utilizada para el blog, permitiendo escribir posts en Markdown con syntax highlighting;
- **Phosphor Icons**: familia de iconos utilizada en el sitio;
- **Geist**: fuente de Google Fonts utilizada en el proyecto.

El sitio está accesible en [otaviorossoni.com](https://otaviorossoni.com) y el código fuente está disponible en [GitHub](https://github.com/Otarossoni/otaviorossoni.com).
