En algún momento, percibí que tener un solo enlace para compartir —algo simple, directo y que reuniera todo en un solo lugar— sería extremadamente útil. Conocía Linktree y herramientas similares, pero quería crear mi propia versión, con mi identidad visual y control total sobre el contenido.

## El Proyecto

La página de enlaces es una aplicación simple en estilo linktree, con todos mis enlaces importantes organizados en un solo lugar: sitio web, blog, redes sociales y correo electrónico.

## Funcionalidades

Las principales funcionalidades implementadas son:

- **Performance**: renderizado estático con Next.js App Router;
- **UI limpia**: Tailwind CSS v4 y fuente Geist;
- **Enlaces organizados**: sitio web, blog, redes sociales y correo electrónico;
- **Foto de perfil**: imagen local optimizada con Next.js Image.

## Tecnologías Utilizadas

Para el desarrollo, utilicé:

- **Next.js 16**: framework React con App Router, responsable del renderizado estático;
- **TypeScript**: lenguaje principal del proyecto, garantizando tipado estático;
- **Tailwind CSS 4**: framework de estilización utilitaria;
- **Geist**: fuente de Google Fonts utilizada en el proyecto.

El sitio está accesible en [links.otaviorossoni.com](https://links.otaviorossoni.com) y el código fuente está disponible en [GitHub](https://github.com/Otarossoni/links.otaviorossoni.com).
