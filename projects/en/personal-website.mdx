Having a personal website was always something I wanted, but for a long time I kept postponing it. When I finally decided to build one, I didn't want just a static page with a generic template. I wanted something that reflected my identity as a developer — minimalist, performant, and multilingual.

## The Project

The site is a personal portfolio with an integrated blog, Markdown support with syntax highlighting, and internationalization for three languages: Portuguese, English, and Spanish.

## Features

The main features implemented are:

- **Internationalization**: support for Portuguese (PT), English (EN), and Spanish (ES) with next-intl;
- **Performance**: static rendering with Next.js App Router + Turbopack;
- **MDX Blog**: posts with GitHub Flavored Markdown, syntax highlighting, and styled typography;
- **Dark/Light mode**: smooth toggle with preference saved in the browser;
- **Clean UI**: Tailwind CSS v4, Phosphor Icons, Geist font.

## Technologies Used

For development, I used:

- **Next.js 16**: React framework with App Router, responsible for static rendering and routing;
- **TypeScript**: project's main language, ensuring static typing;
- **Tailwind CSS 4**: utility-first CSS framework;
- **next-intl**: internationalization library for Next.js;
- **MDX + next-mdx-remote + rehype-highlight**: stack used for the blog, enabling writing posts in Markdown with syntax highlighting;
- **Phosphor Icons**: icon family used on the site;
- **Geist**: Google Fonts typeface used in the project.
