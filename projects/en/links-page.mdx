At some point, I realized that having a single link to share — something simple, direct, and that brought everything together in one place — would be extremely useful. I knew about Linktree and similar tools, but I wanted to create my own version, with my own visual identity and full control over the content.

## The Project

The links page is a simple linktree-style application, with all my important links organized in one place: personal website, blog, social media, and email.

## Features

The main features implemented are:

- **Performance**: static rendering with Next.js App Router;
- **Clean UI**: Tailwind CSS v4 and Geist font;
- **Organized links**: website, blog, social media, and email;
- **Profile photo**: local image optimized with Next.js Image.

## Technologies Used

For development, I used:

- **Next.js 16**: React framework with App Router, responsible for static rendering;
- **TypeScript**: project's main language, ensuring static typing;
- **Tailwind CSS 4**: utility-first CSS framework;
- **Geist**: Google Fonts typeface used in the project.

The site is accessible at [links.otaviorossoni.com](https://links.otaviorossoni.com) and the source code is available on [GitHub](https://github.com/Otarossoni/links.otaviorossoni.com).
